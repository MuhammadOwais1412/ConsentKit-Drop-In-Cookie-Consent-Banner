ConsentKit
Drop-In Integration

One tag.
Fully working.

ConsentKit auto-injects its own CSS and HTML. Add one script tag with data attributes — no copy-pasting, no manual setup, no configuration files.

Zero setup Data attributes GDPR compliant WCAG 2.1 AA 3 themes JS dispatcher
1
Drop in one script tag
Place this before </body>. Everything else is automatic.
Minimum — works out of the box
HTML 
<script src="dist/consentkit.min.js"></script>
Full configuration via data attributes
HTML 
<script
  src="dist/consentkit.min.js"
  data-theme="dark"
  data-position="bottom"
  data-privacy-url="/privacy-policy"
  data-expire-days="365"
  data-auto-show="true"></script>
2
Gate your analytics scripts
Use CookieConsent.on() after the script tag. Callbacks fire only when consent is granted — immediately on load if already given.
JavaScript 
// Place after dist/consentkit.min.js
CookieConsent.on('analytics', function() {
  // Runs only after analytics consent granted
  gtag('js', new Date());
  gtag('config', 'G-XXXXXXXXXX');
});

CookieConsent.on('marketing', function() {
  // Runs only after marketing consent granted
  fbq('init', 'YOUR_PIXEL_ID');
  fbq('track', 'PageView');
});

CookieConsent.on('functional', function() {
  // Runs only after functional consent granted
});

// Chain them
CookieConsent
  .on('analytics', initGA)
  .on('marketing', initPixel);
3
Optional — override via JavaScript
Data attributes cover 100% of use cases. But you can also call CookieConsent.init() to override settings at runtime.
JavaScript 
// Optional — data attributes are enough for most cases
CookieConsent.init({
  theme:      'dark',
  position:   'bottom',
  privacyUrl: '/privacy-policy',
  expireDays: 365,
  autoShow:   true,
});

Data Attributes
data-theme "dark" | "light" | "colorful"
Default: "dark"
data-position "bottom" | "top"
Default: "bottom"
data-privacy-url URL for Privacy Policy link
Default: "/privacy-policy"
data-expire-days Days before consent expires
Default: 365
data-auto-show "true" | "false"
Show banner if no consent stored
Default: "true"
JS API
.on(cat, cb) Gate a script on consent. Fires immediately if already granted.
.show() Show the banner.
.hide() Hide the banner.
.showPreferences() Open the Preference Centre.
.hidePreferences() Close the Preference Centre.
.acceptAll() Accept all categories.
.rejectAll() Reject non-essential categories.
.savePreferences() Save current toggle state.
.getConsent() Returns current consent object or null.
.hasConsent(cat) Returns boolean for a category.
.reset() Clear all consent and re-show banner.
